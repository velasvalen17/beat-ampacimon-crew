# Configuration Files

This folder contains configuration files for the application:

- **crontab** - Cron job configuration for automated daily updates
- **docker-compose.yml** - Docker Compose configuration for local development
- **docker-compose.portainer.yml** - Docker Compose configuration for Portainer deployment
- **stack.yml** - Docker stack configuration (if applicable)

These files are used during deployment and should be reviewed before running in production.
